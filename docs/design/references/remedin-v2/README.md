# Remedin V2 visual references

Estas quatro imagens são as referências oficiais para o redesign acessível do
Remedin V2:

- `home-empty.webp`
- `home-active.webp`
- `add-medication.webp`
- `profile-reminders.webp`

A fidelidade se aplica ao conteúdo dentro da tela do aplicativo. Moldura do
aparelho, status bar e pequenas diferenças de renderização nativa não fazem
parte do critério de comparação.
