# ADR 0001 - Tabs For Primary Navigation

## Status
Accepted

## Decision
Use bottom tabs for Hoje, Histórico, and Configurações. Use stack screens for Adicionar medicamento and Detalhes.

## Context
The app previously exposed Histórico and Configurações through buttons on the Home screen. That made the daily medication flow compete with global navigation.

## Consequences
- Hoje can focus on daily doses and quick actions.
- Histórico and Configurações are always reachable.
- Creation and detail flows remain modal-like stack journeys.
